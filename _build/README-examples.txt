CONTENTS
--------

  - 1xx_getting_started/  Beginner tutorials with PDF guides
  - 2xx_dcdc_converters/  DC-DC converter examples (Buck, Boost, etc.)

  - 4xx_dcac_inverters/   DC-AC inverter examples

  - 6xx_advanced/         Advanced topics

  Real-world application examples.
  - education/            Educational examples from ETH Zurich

articles/
  Circuit files from technical articles and newsletters.

USAGE
-----
Open any .ipes file with GeckoCIRCUITS:
  - Windows:  run-gecko.bat path\to\file.ipes
  - Linux:    ./run-gecko-linux.sh path/to/file.ipes
  - macOS:    ./run-gecko-macos.sh path/to/file.ipes
  - WSL:      ./run-gecko-wsl.sh path/to/file.ipes

Or use File > Open from within GeckoCIRCUITS.
