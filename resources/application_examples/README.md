# Application Examples

Real-world application examples organized by industry sector.

## Categories

| Category | Description |
|----------|-------------|
| [automotive/](automotive/) | Electric vehicle and automotive power |
| [renewable_energy/](renewable_energy/) | Solar inverters, wind power |
| [industrial_drives/](industrial_drives/) | Motor drives and automation |
| [power_supplies/](power_supplies/) | AC-DC and DC-DC power supplies |

## Education Examples

The `education/` folder contains circuit files from ETH Zurich courses:
- `LESI_R*.ipes` - Power electronics course exercises
- `README.md` - Course documentation

## Contributing Examples

To add new application examples:
1. Choose the appropriate category folder
2. Include the `.ipes` circuit file
3. Add a brief README describing the application
