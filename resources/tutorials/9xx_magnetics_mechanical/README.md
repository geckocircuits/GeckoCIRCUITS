## Learning Objectives

- Model magnetic circuits using permeance-capacitance analogy
- Simulate transformer saturation and core losses
- Design inductors with air gaps and saturation effects
- Couple electrical and mechanical domains

## Overview

### Magnetic Domain

The magnetic domain in circuit simulators allows modeling of:
- Transformer saturation effects
- Core losses (hysteresis, eddy current)
- Coupled inductors with leakage
- Non-linear B-H characteristics

### Mechanical Domain

Mechanical modeling enables:
- Motor load dynamics
- Gearbox and friction modeling
- Vibration analysis
- Electromechanical coupling

## Prerequisites

- Complete 2xx DC-DC Converters
- Understanding of magnetic circuit fundamentals
- Basic mechanics (for mechanical tutorials)
