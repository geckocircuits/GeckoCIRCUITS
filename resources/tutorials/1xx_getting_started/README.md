# 1xx - Getting Started

Introduction to GeckoCIRCUITS for new users.

## Learning Path

1. **101 - First Simulation**: Learn to open files, run simulations, and view results
2. **102 - Basic Circuits**: Understand circuit components and wiring
3. **103 - PWM Basics**: Introduction to pulse-width modulation control

## Prerequisites

- GeckoCIRCUITS installed and running
- Basic understanding of electrical circuits
