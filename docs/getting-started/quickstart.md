# Quick Start

Get up and running with GeckoCIRCUITS in 5 minutes.

## 1. Launch Application

```bash
./scripts/run-gecko-linux.sh  # Linux/WSL
scripts\run-gecko.bat          # Windows
./scripts/run-gecko-macos.sh   # macOS
```
