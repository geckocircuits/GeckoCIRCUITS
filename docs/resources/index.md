## Related Tools

| Tool | Description | Link |
|------|-------------|------|
| PLECS | Commercial power electronics simulator (Plexim) | [plexim.com](https://www.plexim.com) |
| PSIM | Power electronics simulation (Altair) | [altair.com/psim](https://altair.com/psim) |
| SIMBA | Modern power electronics with Python API and cloud | [simba.io](https://www.simba.io) |
| LTspice | Free SPICE simulator (Analog Devices) | [analog.com/ltspice](https://www.analog.com/ltspice) |
| QSPICE | Free mixed-signal simulator with SiC/GaN support (Qorvo) | [qorvo.com/qspice](https://www.qorvo.com/design-hub/design-tools/interactive/qspice) |
| Simulink | System simulation (MathWorks) | [mathworks.com/simulink](https://www.mathworks.com/simulink) |
| SIMPLIS | Switched-mode power supply simulator | [simplistechnologies.com](https://www.simplistechnologies.com) |
